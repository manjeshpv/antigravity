# Simple Fund Return Display

This project displays a simple HTML page showing fund return details.

## Requirements Document

### 1. Objective
Create a simple web page to display specific investment return information.

### 2. Content Requirements
The page must display the following text clearly:
- **Fund Name**: "Parag Parikh Arbitrage fund"
- **Return Value**: "5% return"

### 3. Technical Requirements
- **Format**: HTML5
- **Styling**: Clean, centered layout using CSS.
- **Responsiveness**: Basic mobile responsiveness (centered content).
